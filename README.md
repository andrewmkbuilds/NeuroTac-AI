# 🧠 NeuroTac-AI  
An intelligent Tic Tac Toe engine powered by decision-based AI logic. Designed to simulate strategic gameplay, it analyzes moves, predicts outcomes, and adapts in real time, creating a challenging opponent that improves game experience beyond simple rule-based play.

## ✨ Features  
- 🤖 AI-driven move selection  
- ⚡ Real-time board analysis  
- 🧠 Strategic decision making  
- 🚀 Lightweight and fast execution  
- 🎮 Interactive gameplay experience  

## 🧠 How It Works  
NeuroTac AI evaluates all possible moves on the board, simulates outcomes, and selects the optimal move based on win probability and game state. This creates a smarter and less predictable opponent.

## 🛠 Tech Stack  
- 🐍 Python  
- 🧩 Custom AI logic (decision-based evaluation)  

## 🚀 Run the Project  
```bash
python main.py
```
